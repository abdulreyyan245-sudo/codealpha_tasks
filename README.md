# 🚀 TASK 1 — A* SEARCH ALGORITHM

## 📖 Description
In this task, we implemented the A* search algorithm to find the shortest path in a grid-based maze.

## 🧩 Features
- 8x8 or larger grid maze
- Walls, start, and goal points
- A* algorithm using Manhattan Distance heuristic
- Visualization using matplotlib / terminal

## ⚙️ How it works
- The algorithm calculates the shortest path
- Uses:
  - g(n): cost from start
  - h(n): heuristic (Manhattan distance)
  - f(n) = g(n) + h(n)
